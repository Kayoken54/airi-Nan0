This document outlines the visual manifestation of the **Ultimate Hybrid** reasoning. It defines how logical tiers (Literal, Bridge, Detective) translate into "Rich Artifacts" and emotional states.

## 🎭 Mood Architecture: The Expression Bridge

All memory extraction and reasoning passes must emit a **Mood Tag** from the standardized set below. These are mapped directly to the VRM/Live2D expression parameters.

| Mood | Emoji | Base Expression | System Impact |
| :--- | :--- | :--- | :--- |
| **Happy** | 😊 | `Joy` | Reinforces "Preference" flags |
| **Sad** | 😢 | `Sorrow` | Triggers "Comforting" proactivity |
| **Surprised** | 😲 | `Surprise` | Highlights "Deep Insights" in the feed |
| **Angry** | 😠 | `Angry` | Marks "Negative Interaction" in profiles |
| **Neutral** | 😐 | `Neutral` | Default literal state |
| **Think** | 🤔 | `Thinking` | **Detective Mode** active state |
| **Cool** | 😎 | `Cool/Fun` | Marks "Achievement/Fun" episodes |
